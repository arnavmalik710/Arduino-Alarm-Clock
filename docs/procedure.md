# Procedure

1. Connect the DS1307 RTC module to the Arduino UNO using I2C communication pins.
2. Connect the 16×2 LCD display to the Arduino.
3. Connect push buttons for setting time and alarm values.
4. Connect the buzzer to the designated digital output pin.
5. Upload the Arduino program to the Arduino UNO board.
6. Power the circuit and verify the RTC module is functioning correctly.
7. Set the current time and alarm time using the push buttons.
8. Observe the displayed date and time on the LCD.
9. Verify that the buzzer activates when the current time matches the preset alarm time.
